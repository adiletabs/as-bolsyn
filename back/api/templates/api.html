Sections

When we click on one section, they just show the list.
Nobody can edit or add something.
When clicking on a section , it goes to the page with the restaurants of this section.

http://127.0.0.1:8000/api/sections/


List of restaurants

All (authorized and unauthorized) can see the list.
Authorized admins can create and edit the list, others can not.
When you click on one of them, you go to the page with its dishes.

http://127.0.0.1:8000/api/cuisines/pk/restaurants/


Dishes

Everyone sees the list.
Each Dish is tied to one place from the list of categories.
Authorized admins can create and edit dishes, authorized users can add dishes to their order list.
Fields for adding dishes are visible only to authorized users.
Clicking on the 'Reviews' button takes you to a page with restaurant reviews.

http://127.0.0.1:8000/api/cuisines/pk/restaurants/pk/dishes/


Review

Everyone sees the list.
Each Review is tied to one place from the list.
Only authorized users can create new Review.
No one can edit.
The fields for creating a review are visible only to authorized users.

http://127.0.0.1:8000/api/cuisines/pk/restaurants/pk/reviews/


Order

An authorized user sees only his list of orders.
The list of orders is a list of dish and its quantity.
The user can delete a dish and change its quantity.
When you click on the "Confirm" button, the entire list is cleared.
(Plans: bind time series)

http://127.0.0.1:8000/api/orders/
